# Java User Management System

A console-based CRUD application built using Core Java, JDBC, and MySQL.

## Features
- Add new users
- View all users
- Delete users
- Menu-driven console interface

## Technologies Used
- Java (Core Java)
- JDBC
- MySQL

## How to Run
1. Compile:
   javac -cp ".;lib/mysql-connector-j-9.6.0.jar" UserApp.java

2. Run:
   java -cp ".;lib/mysql-connector-j-9.6.0.jar" UserApp
